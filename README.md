# 💰 Controle de Gastos

Aplicação web desenvolvida para gerenciamento de finanças pessoais, permitindo o controle de despesas, cálculo automático de saldo e visualização de dados através de gráficos.

---

## 🚀 Funcionalidades

* ✅ Adicionar despesas com descrição, valor, categoria e data

* ✅ Remover despesas

* ✅ Definir renda mensal

* ✅ Cálculo automático de:
  * Total de despesas
  * Saldo disponível

* ✅ Persistência de dados com **LocalStorage**

* ✅ Visualização de gastos por categoria com gráfico (Chart.js)

---

## 🖥️ Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js

---

## 📊 Preview

O sistema apresenta:

* Interface moderna com cards
